## 2. 美術設計綱領（北極星）

### 2.1 視覺參考
- 《大鬧天宮》（1961 上美廠）：線條的書法骨架
- 宋元水墨小景：留白與意境
- 《中國奇譚》（2023）：當代簡化的線條表現
- 故宮日曆插圖：色彩節制與構圖

### 2.2 視覺不參考
- 日式和風（避免錯位文化）
- 韓式宮廷劇
- 現代 Q 版（避免廉價塑膠感）
- 龍紋金漸層、大量金邊（避免廉價國風）

### 2.3 三大美術原則
1. **留白優先**：書房可以只有一桌、一硯、一窗；庭院多用空地、苔痕、薄霧。
2. **線條為魂**：所有 3D 模型輪廓使用「飛白筆」不規則黑邊，模擬毛筆筆觸。
3. **色彩節制**：主調 **青瓷藍 #6F9DA3、絹黃 #E8D4A2、墨黑 #2A2A2A、朱紅 #C0392B**。
   - 金色 **#D4AF37** 僅用於成就高光時刻。
   - 任何介面元素只能在此色票內。

## 3. 字體解鎖系統（收集物的一部分）

### 3.1 字體進階對照
| 階段 | 漢字頭字體 | 解鎖條件 |
|------|---------|--------|
| 初始 | 標楷體 | 開始遊戲 |
| 府試後 | 行書 | 通過府試 |
| 鄉試後 | 隸書 | 通過鄉試 |
| 會試後 | 篆書（彩蛋）| 通過會試 |
| 高中狀元 | **瘦金體** | 殿試一甲第一名 |

### 3.2 玩家可自由切換已解鎖字體
- 設定面板可選當前頭字體。
- 字體切換時，書生會原地停頓、頭部閃光，配一聲毛筆「沙」音。

### 3.3 介面字體
- 「靈感」「銀兩」「詩集」「赴京趕考」全用手寫書法字（PNG 或 SVG 預渲染），絕不使用系統字。

## 4. 開花動畫（音畫一致）

### 4.1 開花瞬間統一框架
1. 鏡頭推近至花苞，1.5 秒。
2. 背景虛化、四周浮現淡墨「潤開」邊框。
3. 花瓣從中心一筆水墨描邊展開。
4. 對應樂器一聲。
5. 全場景靜默 1 秒。
6. 鏡頭回拉，環境聲恢復。

### 4.2 樂器對照
| 君子 | 樂器 | 情緒 |
|------|-----|------|
| 梅 | 簫 | 清冷孤傲 |
| 蘭 | 古箏一抹 | 幽遠 |
| 竹 | 笛 | 爽朗 |
| 菊 | 古琴一撥 | 恬淡 |

### 4.3 四君子齊開（年度高潮事件）
- 條件：同一個 24 小時內，梅蘭竹菊四盆同時盛開。
- 表現：自動播放完整《梅花三弄》前段 8 秒，鏡頭環繞小院，所有花瓣同時飄落形成「四君子」四個墨字。
- 獎勵：銀兩 +100、稀有種子 ×4、漢字頭可短暫變為「君子」二字交替。

## 5. 天氣的水墨化呈現

| 天氣 | 表現 |
|------|------|
| 雨 | 斜線水墨筆觸從天而落；芭蕉葉抖動；屋簷滴水「答、答」 |
| 雪 | 點墨灑落（白色）；屋簷漸覆白；地面落雪不化 |
| 風 | 紙窗微震；竹葉沙沙；書桌紙頁邊角翻動 |
| 月光 | 窗櫺投影打地板；牆上隱約浮現詩句墨影 |
| 晴 | 陽光斜射，地面有「樹影」漢字輪廓（書生姓氏的飛白） |

## 6. 時辰系統（新增）

### 6.1 時辰對應現實時間
| 時辰 | 現實時段 | 場景表現 | 音景 |
|------|--------|--------|------|
| 卯時（早晨）| 6~10 點 | 日出、薄霧 | 雞鳴、鳥語 |
| 午時 | 10~14 點 | 陽光直照 | 蟬聲 |
| 申時（暮）| 14~18 點 | 夕陽染紅院牆 | 暮鼓 |
| 戌時（夜）| 18~24 點 | 月光、書房點燭 | 燭火搖、遠犬吠 |
| 子時~寅時（深夜）| 0~6 點 | 靜謐、唯研墨呼吸聲 | 沉默 |

### 6.2 深夜彩蛋
- 凌晨上線會看到書生伏案打瞌睡，頭部變灰。
- 玩家點他 → 書生猛抬頭，頭部短暫變「醒」字 0.5 秒，配「啪！」一聲驚醒。
- 此彩蛋有專屬成就「秉燭夜讀」。

### 6.3 時辰影響詩詞題目
- 夜時段玩詩詞遊戲，題目優先抽取「夜」「月」「燭」「眠」相關詩句。
- 暮時段優先「夕」「歸」「鴉」「煙」相關。
- 此設計讓玩家在「對的時辰讀對的詩」，氛圍翻倍。

## 8. 細節氛圍清單（落實玩家三版願望）

- 書桌會堆紙，抄越多詩堆越高，點一下整理。
- 硯台漸生包漿（顏色越深越久）。
- 蘭葉清晨沾露珠，點下會掉。
- 靜置 30 秒 → 鏡頭慢拉至書生背影望月，配遠處琴聲（「文人寂寥」鏡頭）。
- 節氣換景：清明小雨、夏至蓮池、立秋桐落、冬至紅梅、立春嫩芽。

## 9. 與既有 SoundManager 對接建議
