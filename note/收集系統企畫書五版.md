# 《花月》收集系統企劃書 — 五版（最終版）

- 版本：v5.0（最終版；吸收四版反饋並收束全案）
- 撰寫身分：資深遊戲企劃
- 日期：2026-06-17
- 編碼：UTF-8 / 繁體中文
- 副標：**「一人一生的詩路」— 文人模擬器**

---

## 0. 五版定位

> **花月收集系統不是手遊賽季制，而是「華語世界唯一能陪玩家走一生的文人模擬器」。**
>
> 玩家認養一位漢字頭書生，從**入門題詩**（Day 1）到**赴京趕考**（Day 90）、
> 再到**人生分流**（高中後）、最終到**告老還鄉**（累積兩年以上），
> 整個遊戲的長度比一齣戲劇還長，但每一日的單次體驗只需 5~15 分鐘。

---

## 1. 全案三大時間尺度

| 尺度 | 時程 | 內容 |
|------|------|------|
| **每日** | 5~15 分鐘 | 玩 1~3 局詩詞遊戲、澆花、研墨、看時辰彩蛋 |
| **每季** | 約 90 天 | 一次科舉（府試/鄉試/會試），含節氣小考 |
| **每生** | 多年 | 高中後人生分流 → 多年累積 → 告老還鄉 → 開新檔 |

## 2. 主軸玩法（沿用四版，最終定稿）

### 2.1 核心循環
```
玩詩詞遊戲 (game20~35)
  → 詩集 +N 句、靈感、銀兩、種子
  → 灌溉四君子（梅蘭竹菊）
  → 盛開收割 → 銀錠堆入錢箱
  → 節氣小考 → 府試 → 鄉試 → 會試（赴京）
  → 高中或落榜
  → 進入人生分流（翰林 / 邊塞 / 田園 / 江湖）
  → 多年後告老還鄉
```

### 2.2 經濟與數值（沿用三版定稿）

| 項目 | 設定 |
|------|------|
| 上品 / 中品 / 下品 | ×1.5 / ×1.2 / ×1.0 |
| 每日首通加成 | +50% |
| 連續首通 3 局（文思如潮）| +100% |
| 稀有種子掉率（研究所）| 25%，第 5 次保底 |
| 詩集計入方式 | 按句計入 |
| 詩集會試門檻 | 50 句 |
| 學費會試門檻 | 500 兩 |

### 2.3 美術北極星（沿用四版）
- 視覺：大鬧天宮線條 + 宋元水墨留白 + 中國奇譚當代簡化
- 三原則：留白優先 / 線條為魂（飛白邊）/ 色彩節制（青瓷藍、絹黃、墨黑、朱紅；金色僅成就用）

### 2.4 字體解鎖（沿用四版）
標楷 → 行 → 隸 → 篆 → 瘦金（高中狀元彩蛋）

### 2.5 時辰系統（沿用四版）
卯/午/申/戌/子寅五時段 + 凌晨打瞌睡彩蛋 + 時辰影響題庫

### 2.6 開花音畫一致（沿用四版）
梅簫、蘭箏、竹笛、菊琴；四君子齊開《梅花三弄》8 秒。

## 3. 五版新增：書生人格系統（回應四版 B 點）

### 3.1 開局五題人格塑造

新玩家進入遊戲後，先回答五題：

| 題 | 選項 | 影響 |
|----|------|------|
| 你最愛的詩人？ | 李白 / 杜甫 / 王維 / 蘇軾 / 李清照 | 自語詩句傾向 |
| 春雨還是秋雨？ | 春 / 秋 | 場景天氣偏好 |
| 梅還是菊？ | 梅 / 蘭 / 竹 / 菊 | 起始種子 |
| 劍還是琴？ | 劍 / 琴 / 棋 / 書 | 漢字頭副件（佩劍/抱琴/握棋/捧書）|
| 江南還是塞北？ | 江南 / 塞北 / 巴蜀 / 嶺南 | 高中後人生分流預設傾向 |

### 3.2 自語系統
- 書生靜置 30 秒，可能自語一句詩（從玩家詩集 + 人格傾向中抽）。
- 例：選「秋 + 菊 + 琴 + 田園 + 陶淵明」→ 高機率抽「采菊東籬下，悠然見南山」。
- 自語以漂浮的書法字氣泡呈現，3 秒淡出。
- 自語可被玩家「點擊收藏」進「書生語錄」相簿。

### 3.3 人格隨遊戲變化
- 完成府試後可重新調整一項人格。
- 完成鄉試後再一項。
- 高中後人格定型，影響人生路線預設選項。

## 4. 五版新增：人生後篇 — 四條路線（回應四版 A 點）

高中狀元後，玩家可選四條人生路線：

### 4.1 入翰林（朝隱路線）
- 場景：京城翰林院
- 新收集：御賜文房（玉鎮紙、犀角筆）、奏摺典藏
- 詩風主題：王維、白居易、晏幾道（朝中與閒適）
- 長線目標：編成《翰林集》100 卷

### 4.2 出塞為官（豪邁路線）
- 場景：邊塞驛站、烽火臺
- 新收集：戰馬、胡笳、塞外鷹
- 詩風主題：高適、岑參、王昌齡
- 長線目標：守邊三年，集成《邊塞詩鈔》

### 4.3 歸隱田園（恬淡路線）
- 場景：山中草堂、藥圃、小溪
- 新收集：山泉、藥草、白雲、漁網
- 詩風主題：陶淵明、孟浩然、范成大
- 長線目標：耕讀十年，集成《田園詩鈔》

### 4.4 浪跡江湖（俠客路線）
- 場景：行旅扁舟、酒樓、客棧
- 新收集：酒葫蘆、地圖殘片、笛子、行李
- 詩風主題：李白、杜甫（行旅篇）
- 長線目標：踏遍三山五嶽，集成《行旅詩鈔》

> **每條路線各約 3~6 個月內容，主題不重疊，詩詞題庫各異。**

## 5. 五版新增：月選題詩活動（回應四版 D 點）

### 5.1 機制
- 每個月開放一題：例「以『雪』為題」、「以『歸』為題」、「以『燈』為題」。
- 玩家從自己詩集中**集句**回應（使用 game23 縱橫集句系統的技術延伸）。
- 月末前提交，月初公布十首「本月入選」（初期用 AI 評分，後期可開放社群投票）。

### 5.2 獎勵與展示
- 入選者書生頭頂掛「月選」紅綢一個月。
- 入選者作品展示於「翰墨堂」可供所有玩家欣賞。
- 累積 12 次入選 → 漢字頭可短暫變「翰」字。

### 5.3 為什麼這是長線回訪的核心
- 題目永不重複 → 內容無上限。
- 玩家需要先累積詩集 → 強化主玩法。
- 社群展示 → 輕度榮譽感而非競爭排行。

## 6. 五版新增：年度儀式（回應四版 F 點）

### 6.1 除夕煙火與春聯
- 現實除夕日 → 書生在小院燃放煙火，配《茉莉花》全曲。
- 玩家用花月集句系統填寫一副春聯，貼在院門。
- 春聯永久保存於「春聯牆」，第 N 年回看會展示歷年春聯。

### 6.2 元宵燈謎
- 元宵節限定迷你遊戲：燈下猜詩謎。

### 6.3 中秋望月
- 中秋夜書生獨自坐院賞月，配《關山月》古琴曲，鏡頭環繞 8 秒。
- 玩家可拍照分享（不上傳社群亦保存於相簿）。

### 6.4 清明憶舊
- 清明日，若玩家已有「告老還鄉」結局的舊書生，舊書生會出現在新書生小院門口遞一卷舊詩。

## 7. 五版新增：「一生」結局與輪迴（回應四版 E 點）

### 7.1 老去
- 累積遊戲時間 1 年 → 書生留鬚。
- 累積 2 年 → 鬚色泛白。
- 累積 3 年 → 解鎖「告老還鄉」事件。

### 7.2 告老還鄉
- 玩家可主動觸發「告老還鄉」結局。
- 表現：書生收拾行囊，回到最初的小院，所有收集物變為「珍藏館」永久展示。
- 結局動畫：書生背影漸行漸遠，配《送別》。

### 7.3 輪迴與繼承
- 此後可開新檔。
- 老書生會偶爾出現在新書生小院門口（清明、中秋彩蛋），遞一卷舊詩進新書生詩集。
- 此設計讓「玩家自身的時光」成為遊戲的一部分。

## 8. 落榜的優雅處理（回應四版 G 點）

- 落榜不羞辱、不彈出大紅「失敗」。
- 表現：書生默默回小院，案頭多一壺酒，頭部短暫變「醉」字。
- 配杜甫「白也詩無敵」一聲嘆。
- 翌日清晨書生醒來繼續抄詩。
- 玩家詩集多出一頁「落榜詩鈔」，記錄此次趕考期間寫過的詩。
- **失敗也是詩**。

## 9. 輕量社交（回應四版 C 點）

### 9.1 贈詩
- 詩集任一頁可生成「贈某某」卷軸圖片，分享到 LINE/IG。
- 收禮人下載花月時，該頁詩自動入其書架。

### 9.2 鄰院
- 可拜訪好友小院，看花、看詩。
- **不能比較、不能買賣、不能送禮**，僅串門子。
- 串門子可獲得 1 點「友情靈感」/ 日，每日上限 10。

### 9.3 月選翰墨堂
- 全服共享的月選作品展示廳，無排行、無投票競爭性，只展示。

## 14. 結語

**《花月》收集系統，是為了讓玩家「在這個時代仍能擁有一座江南小院」。**

它的數值經得起精算（三版），它的美術經得起內行（四版），
它的長線經得起一輩子的陪伴（五版）。

當玩家第 1095 天打開遊戲，看到頭部已是瘦金體「林」字、白鬚飄拂的老書生
靜靜坐在院中望月，配《關山月》一聲琴 ——
那一刻，這款遊戲就不再是遊戲，而是**他寫給自己人生的一首長詩**。
