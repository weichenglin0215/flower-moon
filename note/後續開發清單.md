# 後續開發清單

本文件記錄已選定的後續開發遊戲，包含每款的介面布局與最簡操作方式。
所有介面以**直版手機**（約 9:16）為設計基準。

操作設計原則：
1. **單擊 > 拖曳 > 多次點擊**
2. 操作複雜度應與「決策數量」匹配
3. 同一款遊戲只使用一種操作方式（同一手勢家族）
4. 玩家從手指碰到畫面到離開，應能完成一個完整決策

---

## A. 收集／消除類遊戲（合成消除 + 詩句收集）

此類遊戲共通設計：畫面上方顯示當前要收集的詩句（每字需收集 N 次），
完成一句後進入下一句，集滿整首詩過關。

### A1. 三消連線版（源自 Candy Crush）

**操作：拖曳交換（單次拖曳）**

```
┌─────────────────────┐
│  關卡 12  進度 1/4   │
├─────────────────────┤
│ 收集目標：床前明月光 │
│  床:●●○ 前:●○○      │
│  明:○○○ 月:●●●✓     │ ← 字的收集進度
│  光:○○○             │
├─────────────────────┤
│  ┌──┬──┬──┬──┬──┐  │
│  │床│明│月│前│光│  │
│  ├──┼──┼──┼──┼──┤  │
│  │明│月│床│光│前│  │ ← 字塊棋盤
│  ├──┼──┼──┼──┼──┤  │   7x7 或 8x8
│  │光│前│月│床│明│  │
│  ├──┼──┼──┼──┼──┤  │
│  │…│…│…│…│…│  │
│  └──┴──┴──┴──┴──┘  │
├─────────────────────┤
│  💡提示  🔄洗牌      │
└─────────────────────┘
```

**操作流程**：
- 手指按住一個字塊 → 朝上下左右拖曳 → 放開時與相鄰字塊交換
- 若交換後形成同字三連，即消除並累加該字的收集進度
- 無法形成三連則交換還原

### A2. 物理掉落版（源自 Bejeweled / Puzzle & Dragons）

**操作：拖曳路徑（單次拖曳）**

```
┌─────────────────────┐
│  關卡 5  連鎖 ×3     │
├─────────────────────┤
│ 收集目標：春眠不覺曉 │
│  春:●●● 眠:●○○      │
│  不:○○○ 覺:○○○ 曉:○○○│
├─────────────────────┤
│  ┌──┬──┬──┬──┬──┐  │
│  │春│眠│不│覺│曉│  │
│  ├──┼──┼──┼──┼──┤  │
│  │覺│春│曉│眠│不│  │ ← 棋盤
│  ├──┼──┼──┼──┼──┤  │   手指拖曳一個字
│  │曉│不│覺│春│眠│  │   穿過同字路徑
│  ├──┼──┼──┼──┼──┤  │
│  │…│…│…│…│…│  │
│  └──┴──┴──┴──┴──┘  │
├─────────────────────┤
│  ⏱剩餘 5 秒         │
└─────────────────────┘
```

**操作流程**：
- 手指按住一個字塊 → 不離開螢幕，拖曳路徑經過相鄰同字字塊（含斜向）
- 拖曳過程顯示路徑連線高亮，所經過的同字字塊發光
- 放開手指 → 若路徑長度 ≥ 3，同字消除並累加收集，未消除字塊重力下落補位
- 連鎖反應（補位後又形成同字 3 連）自動觸發倍率加成

```
├─────────────────────┤
│   ○○○○○○○         │
│    ○○○○○○          │ ← 字泡泡牆
│   ○○○○○○○         │   由上往下推

│                     │
│         ╱           │ ← 瞄準虛線
│        ╱            │   隨拖曳方向
│       ╱             │
├─━━━━━━━━━━━━━━━━━─┤
```

- 命中相同字 3 顆以上即消除並累加收集
- 失敗條件：字牆觸及底部壓力線

### A4. 俄羅斯方塊版（源自 Tetris）

```
┌─────────────────────┐
│  關卡 3  消除行 12   │
├─────────────────────┤
│ 收集目標：登鸛雀樓   │
│  白:●○○ 日:○○○ …   │

│  ┌──┬──┬──┬──┬──┐  │
│  │  │  │白│日│  │  │ ← 下落中方塊
│  ├──┼──┼──┼──┼──┤  │   每塊含 2~4 個字
│  │  │  │  │  │  │  │
│  ├──┼──┼──┼──┼──┤  │
│  │  │依│山│  │  │  │ ← 已堆疊字
│  ├──┼──┼──┼──┼──┤  │
│  │白│日│依│山│盡│  │
```

- **左滑／右滑**：水平移動下落方塊

- 方塊堆疊後，同字橫向或縱向 3 連即消除並累加收集
- 失敗條件：堆疊觸頂

```
│       ╲▓▓▓▓▓╱      │   可旋轉視角
│                     │
├─────────────────────┤
│  暫存槽（最多 7 個） │
│  ┌─┬─┬─┬─┬─┬─┬─┐  │
│  │紅│相│ │ │ │ │ │  │ ← 點過的牌
```

- 失敗條件：暫存槽滿 7 張仍無法湊出 3 同字
- 視角操作：雙指旋轉 3D 視角找牌（屬於檢視操作，不算遊戲決策手勢）

## B. 益智／認知類遊戲（學習導向）

此類遊戲共通設計：低門檻、無需事先會背詩，過關即累積詩集收藏。

### B1. 詩眼覓蹤 — 字詞替換偵探

**操作：拖曳（同時決定「哪個字錯」+「正確答案」）**

```
┌─────────────────────┐
│  關卡 12 ❤❤❤ ⏱30s   │
├─────────────────────┤
│                     │
│   (拖曳時直排候選)   │
│       霜            │
│       露 ← 對齊線   │
│       雪            │
├─━━━━━━━━━━━━━━━━━─┤
│   春眠不覺曉         │
│   處處聞啼鳥         │ ← 詩句橫排
│         （某字被換） │   中央偏下
├─────────────────────┤
│  💡提示 ⏸暫停 ⏭跳關 │
└─────────────────────┘
```

**操作流程**：
- 手指按住懷疑被替換的字 → 往下拖曳
- 該字上方浮現直排候選字，隨拖曳距離捲動
- 對齊紅線位置者即為當前選擇
- 放開手指 → 提交答案，電腦判定
- 答對：替換動畫＋煉字典故；答錯：該字滲血碎裂，扣生命

```
│   │   ●洛陽     │   │

├─────────────────────┤
│   ┌─────────────┐   │
│   │ 故事說明卡   │   │
│   │ 李白遊廬山…  │   │
│   └─────────────┘   │
```

- **階段二**：讀完故事，點擊詩文中發光的關鍵字（地名／景物）→ 該字飛出標到地圖對應位置 → 全部標完過關
- 每完成一處地點，地圖上插旗紀錄收藏
