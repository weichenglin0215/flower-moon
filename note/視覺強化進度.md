# 視覺強化進度追蹤（game24~game35）

> 目標：把 game24~game35 從「能跑」升級到「優秀經典益智遊戲級」的視覺體驗。
> 策略：逐一強化，每款獨立 agent 處理，session 中斷時自動排程續跑。

## 強化重點（所有款式共通）

### 1. 配色與字體一致性（依「遊戲類型介面設計與程式碼規範」§5）
- 主字體：`'Noto Serif TC', serif`（思源宋體）
- 強調色：`hsl(36, 80%, 50%)`（金橙）
- 正確：`hsl(145, 68%, 25%)`（深翠綠）/ 反饋色 `hsl(145, 68%, 36%)`
- 錯誤：`hsl(0, 70%, 25%)`（朱紅）/ 紅心 `hsl(4, 90%, 60%)`
- 背景徑向漸層：中心 `hsl(210, 30%, 25%)` → 邊緣黑
- 字體尺寸統一以 px、不可用 rem

### 2. 點擊反饋（按鈕、字塊）
- 按下瞬間：scale-down 0.92 + 短暫亮度提升
- 放開瞬間：scale 回彈到 1.0 + 過渡 120ms
- hover：`filter: brightness(1.5)` + `box-shadow: 0 0 8px rgba(255,255,255,0.9)`
- 觸控 `:active` 必須有反饋（不只 hover）

### 3. 動畫節奏感（最關鍵）
- 消除類：粒子爆破、字魂飛升、連鎖時序錯落
- 掉落類：彈跳緩衝（cubic-bezier overshoot）、新生字塊 pop-in
- 移動類：平滑 transition 而非瞬移
- 連擊：大型放大字浮現、CSS keyframe + 隨時間消失
