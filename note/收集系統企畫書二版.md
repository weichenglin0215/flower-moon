## 1. 系統定位（修訂）

> 一位漢字頭的書生，在江南小院中養花、抄詩、煮茶，
> 以詩詞遊戲累積的「靈感」灌溉四君子（梅蘭竹菊），
> 待四君子皆盛開、詩集滿百，便可束裝赴京趕考。

---

## 2. 主軸：四君子盆景養成（Primary Loop）

### 2.1 為什麼是四君子
- **詩詞濃度極高**：梅蘭竹菊在唐宋詩詞中是常客（王安石〈梅〉、鄭板橋〈竹〉、陶淵明〈飲酒〉之菊…）
- **文化人格映射**：梅=傲、蘭=幽、竹=堅、菊=逸，玩家可選性格傾向
- **視覺最美**：開花動畫即遊戲最大爽點
- **節氣天然契合**：梅冬、蘭春、竹夏、菊秋

### 2.2 養成五階
種子 → 抽芽 → 抽枝 → 結蕊 → 盛開
（每階對應 3D 模型變化 + 開花時鏡頭推近 + 簫音一聲）

### 2.3 每盆需求
- 澆水：每日一次（不澆不會死，只是慢一天）
- 詩詞灌注：玩對應主題詩詞遊戲，注入「靈感」加速生長
- 修剪：影響花型，玩家可培育出特殊品種（如雙鉤蘭、墨梅、五瓣菊）

### 2.4 上品/中品/下品
- 在「盛開」階段的前 24 小時內收割 → 上品（學費 ×3）
- 24~72 小時內收割 → 中品（學費 ×1.5）
- 超過 72 小時 → 下品（學費 ×1，但仍可收割，不會枯萎消失）

---

## 3. 次要循環（Secondary Loops，作為點綴）

> 全部「可選」，非強制每日。

### 3.1 抄書（書卷收藏）
- 每首詩詞遊戲首次通關 → 自動抄入「我的詩集」一頁
- 詩集滿百 → 解鎖「赴京資格」
- 書架上的卷軸數量可視覺累積

### 3.2 研墨（資源類）
- 用於抄書與灌注盆景
- 每日小遊戲：毛筆繞硯三圈（5 秒）獲得墨水
- 不做也行，可用銀兩買墨

### 3.3 烹茶（增益類）
- 玩詩詞遊戲前可選擇沏一壺茶
- 提供當局「降難度」「延長時間」「重試一次」三選一
- 屬於消耗品，不是收集

---

## 4. 核心循環圖

```
玩詩詞遊戲(game20~35)
     ↓
首次通關 → 詩集 +1 頁、靈感 +高額、隨機種子掉落
重玩通關 → 靈感 +少額
     ↓
靈感澆灌盆景 → 加速生長
     ↓
盆景盛開 → 收割 → 銀兩
     ↓
銀兩 = 學費
     ↓
詩集滿百 + 學費滿 500 兩 → 赴京趕考
```

## 5. 學費經濟與趕考節奏（修訂）

### 5.1 三級科舉路線
| 級別 | 條件 | 時程 | 獎勵 |
|------|------|------|------|
| 府試 | 詩集 ≥ 20 首 + 銀兩 ≥ 50 兩 | 約 14 天 | 解鎖蘭花稀有品種、墨工具升級 |
| 鄉試 | 詩集 ≥ 50 首 + 銀兩 ≥ 200 兩 | 約 45 天 | 解鎖畫案、書畫收藏 |
| 會試（赴京趕考）| 詩集 ≥ 100 首 + 銀兩 ≥ 500 兩 | 至少 90 天 | 殿試動畫、狀元/榜眼/探花/進士分支結局 |

### 5.2 結局可重啟（春闈三年）
- 落榜不結束，回小院繼續累積，下次趕考有「考過經驗」加成。
- 高中後可選「告老還鄉」開新檔，繼承部分稀有收藏。

## 6. 環境與氛圍（強化）

### 6.1 聽覺設計
- 背景：風吹竹葉、紙窗微震、遠處古琴片段（《平沙落雁》《漁樵問答》《梅花三弄》隨機）。
- 工作音：研墨「沙沙」、毛筆「沙沙」、剪枝「咔擦」、澆水「淅瀝」。
- 開花音：盛開時一聲幽簫，鏡頭推近，全環境瞬間靜默。
- 整合 SoundManager：花開時呼叫 `melodyPlayer.playFullMelody('梅花三弄片段')`。

### 6.2 視覺氛圍
- 動態天氣：晴/陰/雨/雪，影響盆景生長係數。
- 月光：夜間遊玩時，月光從窗櫺斜照書桌。
- 雨打芭蕉：下雨時院角芭蕉葉上有雨滴特效。
- 紙鳶彩蛋：偶爾天空飄過紙鳶（古代孩童玩具，非現代）。

### 6.3 漢字頭擴充
| 觸發 | 表現 |
|------|------|
| 喝茶/酒 | 頭部微微泛紅 |
| 詩成 | 頭部 3 秒變成詩中關鍵字（「月」「梅」「雪」） |
| 下雨外出 | 頭頂自動戴竹笠 |
| 思考受阻 | 頭頂浮現「？」漢字 |
| 達成成就 | 頭部變金 + 衣袖飄絲帶 |

## 7. 每日回訪設計（無壓力版）

- **每日簽到**：到書桌前喝一口茶，得一點靈感（30 秒可完成）。
- **節氣彩蛋**：當日專屬詩詞題與限定種子。
- **無強制懲罰**：盆景不會凋謝，只是錯過上品時機。
- **進度視覺化**：院門口貼出「距春闈尚有 XX 日」紅紙。

---

## 8. 與詩詞遊戲串接（精修）

- 每款遊戲根據主題綁定**主要君子**：
  - 詠梅題 → 灌溉梅盆
  - 詠竹題 → 灌溉竹盆
  - 詠蘭、詠菊類推
  - 非詠物題 → 玩家自選哪盆受益
- 高難度（研究所）通關 → 機率掉落稀有種子（如「墨梅」「雙鉤蘭」）。
- 連續首次通關 3 首 → 「文思泉湧」狀態，1 小時內所有資源 +50%。

---

## 9. 3D 場景修訂

- 場景縮小為「一進院落」：影壁、天井、廂房、書房、花圃，視角更聚焦。
- 鏡頭在「盛開」時自動電影化（推近、慢鏡、配簫音）。
- 操作仍為點地行走 + 點功能區互動。

---

## 10. 風險與待解

1. 「研究所掉率」要避免讓人為刷高難度而痛苦。
2. 「90 天會試」要靠詩集進度視覺化來維持期待感。
3. 詩集滿百是否太高 — 待玩家二輪反饋確認。
