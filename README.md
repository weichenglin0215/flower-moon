# 花月 Flower Moon - 唐詩宋詞學習遊戲平台

![花月 Logo](images/FlowerMoon_512.png) 

## 專案簡介
**花月 (Flower Moon)** 是一個致力於推廣**唐詩宋詞**的互動式網頁平台。我們的核心目標是讓詩詞愛好者在輕鬆愉快的遊戲氛圍中學習經典文學，深入感受中國傳統文學的韻律與意境之美。

透過整合日曆、卡片記憶與多樣化的趣味遊戲，我們將深奧的古典文學轉化為觸手可及的互動體驗，讓讀詩、背詩、品詩不再是枯燥的任務，而是一場跨越時空的文化探險。

## 主要頁面與功能介紹

### 📅 詩詞日曆 (Main Page)
這是平台的門面，結合了中國傳統曆法與詩詞：
- **每日一詩**：每天推薦一首精選唐詩或宋詞，陪你度過春夏秋冬。
- **農曆與宜忌**：整合農曆資訊與民俗「宜忌」，展現深厚的傳統文化底蘊。
- **節日提醒**：自動標記重要節慶，並提供與時令相符的文學內容。

### 🌙 隨遇而安（視覺療癒頁）
一頁無計分、無勝負的純觀看舒壓互動頁面，取多首詩詞讓文字自行漂移聚散：
- **多詩混拆湊字數**：為了讓畫面熱鬧、字數夠多，改為「多首詩詞湊字數」——每次向題庫抽一小段固定偶數句（至少 2 句、符合「兩句一聯」原則）的詩句，逐段累加不同首詩詞，直到全部字數落在目標範圍，絕不會只抽到單一句或奇數句。
- **拆字散落**：把抽到的所有句子逐字拆開，隨機灑落於深色畫面中，點擊「隨遇而安」按鈕才開始漂移。
- **物理漂移與碰撞**：每個字各自以不同速度、方向緩緩漂移並略為減速；碰到畫面邊緣會反彈並略微加速，同時附加隨機偏轉角度，避免陷入完全水平/垂直、永遠碰不到其他字的死路徑。
- **手指/滑鼠拖曳擾動**：可直接用手指或滑鼠拖曳畫面，半徑範圍內的字會被主動推開——尚未開始漂移時也有即時位移回饋，開始漂移後拖曳還會附帶一點延續動能，讓玩家能主動撥弄字塊互動，不只是被動旁觀。
- **同句相鄰合併成句**：兩字相碰時，若屬同一句的相鄰位置即合併成一組（尺寸隨每次有效合併漸漸放大），繼續以同樣規則漂移碰撞；若不屬同句相鄰位置則彈開並瞬間加速。句子之間永不合併，各自獨立漂移。
- **依完成順序著色**：每句字首次合併（湊滿 2 字以上）即依「第幾句完成連接」的順序著色（紅、橙、黃…依句數平均分布色相），句子完全拼成後仍持續漂移碰撞。
- **碰撞特效與音效**：字相碰時瞬間出現貼合字塊實際長寬的白色（有效合併時改為該句顏色）圓角粗邊框，並播放碰撞音效——無效碰撞短促、有效合併音階較高且至少 1 秒漸弱消失。
- **换詩重來**：全部句子拼齊後，畫面依然持續漂移碰撞，並再度顯示「隨遇而安」按鈕，點擊即重新混拆多首詩詞、重新灑落。

### 🧧 詩詞珠簾（珠簾詩 / 舒壓頁）
一頁無計分、無勝負的純舒壓互動頁面，靈感取自傳統珠簾：
- **匾額顯示玩家文位**：畫面上方橫樑掛的匾額文字即為玩家目前文位（書僮～大儒），依文位高低決定垂掛的珠簾條數（書僮＝1 條，大儒＝15 條），激勵玩家持續累積積分。
- **同首詩成串**：每條珠簾皆由同一首詩的連續詩句串成（句間「，」、句末「。」），長度 10～32 字不等，長短交錯；每次開啟重新隨機抽詩、隨機取字數。
- **Verlet 珠串物理**：以繩索物理模擬珠子上下相連擺動，撥弄手勢僅牽動同一條珠簾（左右各條互不碰撞、互不干擾），每條各自隨機的重力與阻尼參數，讓擺動節奏各不相同。
- **自然吹拂效果**：微風力道隨深度（越靠末端）遞增並帶行進波相位差，使珠簾自然彎成 S 形曲線而非直線傾斜；另有不定時「橫掃陣風」效果，風鋒在 0.5～2 秒內平滑掃過整個畫面（由左至右或右至左），沿途珠簾依序被吹動，而非同時瞬間抖動。
- **雙指縮放/平移**：支援手勢縮放（0.5x～3x）與雙指平移，方便近看細讀或退遠欣賞整片珠簾。
- **文位鼓勵語**：文位在「童生」（含）以下時，畫面左右兩側各加掛一條鼓勵珠簾「提升文位可以增加串珠的數量，加油吧！」，達到縣案首以上自動移除。

### 🌌 文字雲（Poetry Cosmos）
一套以「詩人聲望數據」驅動的三層沉浸式 3D 導覽介面，取代傳統列表選單：
- **朝代 → 詩人 → 詩詞**：三層費波納契球面佈局，字體大小依累積詩詞評價聚合（朝代＝旗下詩人評價總和、詩人＝旗下詩詞評價總和），球面正面中心恆為評價最高者；最高與最低評價項目的尺寸差距調校為 8 倍（2.0x～0.25x），加大鏡頭廣角使前後透視差達 2.5 倍，讓球面立體感更明顯。
- **無萬向鎖的自由旋轉**：以旋轉矩陣（非歐拉角）控制鏡頭朝向，拖曳可任意方向連續轉動、不卡極點；往上拖曳＝球體往上轉（符合直覺）；星空背景與文字球共用同一旋轉矩陣同步轉動；靜止時完全靜止，不自轉、不脈動，僅拖曳/慣性才轉動。
- **兩段式點擊**：第一次點擊鎖定預覽（白框置中），第二次點擊才確認鑽入下一層，避免密集球面誤觸；鎖定狀態在拖曳轉動鏡頭期間持續維持（白框、米色關聯線、米色虛線框不因轉動而消失）。
- **藍色球面線框 + 米色關聯提示線**：相鄰項目間以藍線相連構成一顆完整球體，強化空間感；跳過中間層瀏覽時，鎖定項目與同源項目（如同朝代詩人）間另浮現米色連接線，兼顧效率與知識脈絡。
- **球面防重疊佈局**：依評價與字數估算每個項目所需的角空間，透過力導向鬆弛演算法自動排開，避免密集詩詞標題互相重疊；標題超過 5 字自動截斷為「⋯」。
- **雙勾選框控制深度**：可自由選擇從朝代、詩人或直接以全部詩詞開始瀏覽；縮放過小自動返回上一層（跨層級 radius 差異已修正，各層皆可靠觸發）。
- **資料瀏覽頁面互斥群組**：與成就紀錄、群英榜、江南小院、名人列傳同屬一組，開啟任一頁面或遊戲時自動關閉其餘同組頁面；z-index 與名人列傳同層，「詩詞資料」彈窗能正確蓋在文字雲之上。
- 點擊任一詩詞可直接開啟既有的「詩詞資料」詳細頁面。

### 🏛️ 考棚（Exam / v0.28.0）
